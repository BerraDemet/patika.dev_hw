HTML Nedir?

İşaretleme dili olan Html, web sayfalarının hazırlanmasında kullanılan sistemdir. Bir programlama dili olmayan Html bilgisayarlarımızda kullandığımız web sitelerinin oluşturulmasında kullanılır. Chrome, Fİrefox ve İnternet Explorer gibi tarayıcılar html kodlarını işleyerek bu kodları web sayfasına dönüştürür.

HTML hakkında:

  • Kodlara, “tag” yani etiketler ile komut verilir.
  • Etiketler, “< >” şeklindeki küçüktür ve büyüktür işaretleri arasına yazılır.
  • Etiket açmak için gerekli kod, “< >” işaretleri arasına yazılır ve açılan etiketin kapatılması gerekir.s

CSS Nedir?

Cascading Style Sheets (Basamaklı Stil Şablonları ya da Basamaklı Biçim Sayfaları, bilinen kısa adıyla CSS), HTML'e ek olarak metin ve format biçimlendirme alanında fazladan olanaklar sunan bir işaretleme dilidir.

CSS’in Versiyonları

  • CSS’in ilk çıkışı sürümü CSS1 olarak tanımlanmaktadır.
  • 1992 CSS1’e yeni yetenekler kazandırılarak CSS2 sürümünü çıkardı.
  • 1999 Yılında da CSS3 taslakları yayınlandı.
  • CSS4, tam anlamıyla bir sürüm olarak tanımlanmasa da, CSS3'ün genişletilmiş ve geliştirilmiş hali olarak düşünülebilir.

JavaScript Nedir?

JavaScript Nedir? - JavaScript'e (JS) Ayrıntılı Bakış - AWS JavaScript, geliştiricilerin etkileşimli web sayfaları oluşturmak için kullandığı bir programlama dilidir. JavaScript işlevleri, sosyal medya akışlarını yenilemekten animasyonlar ve etkileşimli haritalar göstermeye kadar, bir web sitesi kullanıcısının deneyimini iyileştirebilir.

JavaScript Tarihi

  • 1995 - Brendan Eich tarafından Netscape için JavaScript geliştirildi.
  • 1996 - JavaScript, Netscape Navigator tarayıcısında kullanıma sunuldu.
  • 1997 - ECMA tarafından ilk JavaScript standardı (ECMAScript 1) yayımlandı.
